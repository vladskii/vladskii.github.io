My Time at KASI Insights

I had a wonderful time working at KASI Insights as a data science intern and Yannick(CEO) really kick-started my love for statistics and programming by providing me with this opportunity. The various tasks completed included:
  • Automated indexing for datasets
    • Here I created a program that takes raw datasets as input and autonomously outputted a menu of all the given parameters (columns) with a reference from which dataset it belonged to and which other datasets it is found within.
    • This was done for the sake of organizing legacy data.
  • Automated index calculator
    • This was the main task for my intern period and was most interesting.
    • Based on all the possible datasets and new incoming datasets, this program calculated certain consumer indexes for 7 different countries as well as a generalized total for the continent.
    • This task was especially difficult as the legacy data required extensive cleaning and wrangling for legitimate results.
  • All work at KASI Insights has been completed using Python with NumPy and Pandas libraries.
